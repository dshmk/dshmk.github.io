Vue.jsのプロジェクトを立ち上げてgitにpushするまでのメモ

Date [ 12 Mar 19 00:24 JST ] Categories [ web ] Tags [ #vue.js ]

数カ月前にVue.jsをやって、時間を空けたら脳内リセットされていた。
ちゃんとメモる。

前提

node --version
# v8.9.4
npm --version
# 5.6.0
sudo npm install -g @vue/cli
vue --version
# 3.5.0

Vue.jsのプロジェクトを作成する

今回はビデオアノテーション用のプロジェクトなので、そんな感じの命名でいく。

vue create video-anno-web

なにも考えず、デフォルトを選ぶと勝手にセットアップしてくれるっぽい。
ご丁寧にREADME.mdまで作られている。

REAMEに従って、残りのセットアップとhot-reloadsを起動する。

npm install
npm run serve

とりあえず、プロジェクトの雛形ができた。

Gitリポジトリにコミットする

なぜか、git commitを打つ必要がなかった…。なぜだ。
とにかくVue.jsがgitで管理しやすいように.gitignore含めて良しなにやってくれている。

git init
git remote add origin https://github.com/dshmk/video-anno-web.git
git add .
git push origin master
